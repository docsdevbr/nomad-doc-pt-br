# Documentação do Nomad

O Nomad é um orquestrador de cargas de trabalho flexível para implantar e
gerenciar qualquer aplicação em contêiner ou legada usando um fluxo de trabalho
simples e unificado.
O Nomad pode executar diversas cargas de trabalho, incluindo Docker, aplicações
não conteinerizadas, microsserviços e em lote.

Se você está apenas começando a usar o Nomad, consulte a
[Coleção de tutoriais de introdução][gs].
