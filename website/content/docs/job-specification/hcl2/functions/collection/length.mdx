# `length` Function

`length` determines the length of a given list, map, or tuple.

If given a list, map, or tuple, the result is the number of elements in that
collection.

## Examples

```shell-session
> length([])
0
> length(["a", "b"])
2
> length({"a" = "b"})
1
```
