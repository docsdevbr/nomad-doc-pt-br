# `trimspace` Function

`trimspace` removes any space characters from the start and end of the given
string.

This function follows the Unicode definition of "space", which includes
regular spaces, tabs, newline characters, and various other space-like
characters.

## Examples

```shell-session
> trimspace("  hello\n\n")
hello
```

## Related Functions

- [`chomp`](/nomad/docs/reference/hcl2/functions/string/chomp) removes just line ending characters from the _end_ of
  a string.
