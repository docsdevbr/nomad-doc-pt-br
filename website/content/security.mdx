# Segurança do Nomad

Entendemos que muitas pessoas usuárias depositam grande confiança na HashiCorp e
nas ferramentas que desenvolvemos.
Aplicamos as melhores práticas e focamos na segurança para garantir que possamos
manter a confiança da comunidade.

Agradecemos profundamente qualquer esforço para divulgar vulnerabilidades com
responsabilidade.

Se você deseja relatar uma vulnerabilidade, consulte a
[página de segurança da HashiCorp](https://www.hashicorp.com/security), que
possui o e-mail apropriado para comunicação, bem como nossa chave PGP.
Por favor, **não crie uma issue no GitHub para relatar questões de segurança**.

Se você não estiver relatando uma vulnerabilidade sensível à segurança, abra uma
issue no repositório padrão do [GitHub](https://github.com/hashicorp/nomad).
